# Goodreads Book Data Analysis 

# Project Owner: 
Katelyn Stanton

# Project Overview 
This semester-long project that applies data mining techniques to a Goodreads Book Dataset. The project will analyze a dataset from Goodreads (May 2024) that contains books along with detailed attributes to anser three discovery questions: 
1. Which book attributes are associated with high reader engagement?  
2. Which books receive unusually high or low ratings relative to their genre?  
3. What patterns in productivity and book quality distinguish successful authors from struggling authors?
